This project focuses on predicting crop yield using machine learning techniques based on various agricultural and environmental factors such as weather conditions, soil properties, rainfall, temperature, and crop type. The main goal is to help farmers and agricultural planners make data-driven decisions to improve productivity and optimize resource usage. The system uses historical datasets to train multiple regression models and identifies the most accurate model for yield prediction. It also provides insights through visualizations, helping users understand how different factors influence crop production. This project is designed as a web-based application (Flask-based) where users can input required parameters and get predicted crop yield results instantly. It supports model evaluation, performance comparison, and result visualization for better interpretability.
